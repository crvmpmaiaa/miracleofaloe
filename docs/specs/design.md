**Author:** maiaa.ai (via brainstorming session)

## Context

Maiaa.ai is pitching Miracle of Aloe (miracleofaloe.com — a 40-year family-owned aloe skincare brand, Dallas-based, distributed in Walmart and Walgreens, 1,052 reviews / 4.9★ on Bay Reviews) a Shopify rebuild. The proposal offers two tiers: **Foundation ($5,950 / 3 wks)** and **Signature ($9,900 / 5 wks)**.

This spec covers a **sales-pitch demo** — a homepage and a single product landing page — built in the existing `native-nature/` scaffold, populated with Miracle of Aloe branding. The demo exists to sign the client by showing them the visual leap from their current site to the one being proposed.

The related proposal document is `maiaa_proposal_miracle_of_aloe (2).html` (supplied during brainstorming, not in repo).

## Goals

1. **Close the pitch.** The demo must visibly out-class miracleofaloe.com and make the proposal's promises feel tangible.
2. **Visually demonstrate the Signature tier.** Custom homepage + custom PDP, considered motion, editorial brand story woven through.
3. **Blur up toward Bespoke.** Include two GSAP-choreographed cinematic moments and an interactive UltraAloe® brand-story sequence — features nominally from the Bespoke tier — to maximise wow. Positioning during the pitch will qualify scope.
4. **Prove conversion discipline.** The PDP in particular must feel like it was designed by someone who understands how DTC landing pages convert, not just how they look.
5. **Make asset generation achievable.** All cinematic / editorial assets are to be generated by the user via Nano Banana Pro (stills) + Seedance (video) per the pipeline defined in `SKILLassets.md`. Placeholder slots in the build must be clearly marked with exact dimensions, subject, and motion directives.

## Non-Goals

- No real checkout, auth, subscription backend, or reviews API — UI is simulated for the demo.
- No About / Ingredients / FAQ / Journal pages — nav links either scroll to homepage anchors or go to `#`.
- No PDPs for products other than Foot Repair Cream.
- No Shopify migration work — this is a static HTML/CSS/JS demo.
- No schema validation for Shopify production use — any schema included is cosmetic for demo.
- No Texas / Dallas / regional references anywhere — positioning is deliberately broad/national.

## Design System

### Palette

| Role | Hex | Use |
|---|---|---|
| Paper | `#F6F2EA` | Default bg, warm cream with film-grain |
| Ink | `#1A1F1C` | Body text, near-black with green undertone |
| Forest | `#0F3B2C` | Full-bleed dark sections (the "dark paper" moments) |
| Aloe | `#8DC735` | Accent energy, micro-highlights (from client brand) |
| Jungle | `#00965E` | Primary CTA background (from client brand) |
| Mist | `#D8E4D1` | Soft section tints, dividers, hover states |
| Muted | `#6B6E63` | Secondary text, labels, metadata |

### Typography

- **Display serif:** Instrument Serif (Google Fonts) — tall, elegant, free, editorial feel
- **Body sans:** Inter (Google Fonts) — neutral, screen-perfect workhorse
- **Detail mono:** JetBrains Mono (Google Fonts) — used sparingly on section labels, price tickers, apothecary captions

### Material / Texture

- SVG film-grain overlay at ~6% opacity on paper sections
- 24px default card radius; 999px pill for buttons
- No drop shadows except on products during motion
- Generous whitespace (sections default ~160vh of vertical breathing room where appropriate)
- Display type capped at ~14 words per line

## PDP (Foot Repair Cream) — 10 Modules

1. **Above-the-fold product card**
   - Left: editorial jar hero + thumbnail gallery (product angles from miracleofaloe.com)
   - Right: breadcrumb (Home / Foot & Hand / Foot Repair) · 4.9★ with review count · product name in serif · price ($36.99) · quantity selector · **Add to Bag** (becomes sticky on scroll past) · subscription toggle (simulated) · free shipping inline note

   - Serif lede: *"It's not just dry. It's the kind of cracked skin that catches on socks, aches by evening, and never seems to get better on its own."*
   - One supporting image; warm paper bg
3. **Before / after** — dual image module, "Day 1 → Day 14." Either simple side-by-side or a draggable slider.
4. **The formula — ingredient-by-ingredient** — 4 ingredient cards: Aloe Vera (UltraAloe®) · Vitamin E · Allantoin · Shea Butter. Each card: ingredient still + name + one-line benefit (sans body copy).
5. **Why UltraAloe® works** — small cinematic moment reusing the homepage Moment 2 stills; 40-year formula story in 3 lines; forest-green section break from surrounding paper
6. **How to use** — 3-step module with lifestyle imagery: *Cleanse · Apply · Repeat.* Minimal copy, editorial photos.
7. **FAQ** — accordion, 6 real questions: "How fast does it work?" · "Is it greasy?" · "Safe for diabetics?" · "What's UltraAloe®?" · "Can I use daily?" · "Is it vegan?"
8. **Review wall** — 8 photo reviews + filterable tags (Dry heels · Cracked feet · Diabetic skin · Seasonal dryness)

10. **Final CTA + guarantee** — full-width closer: "100% satisfaction or your money back." Final Add to Bag. Sticky ATC remains visible throughout all modules below the fold.

## Technical Approach

### Stack

- HTML5 + CSS3 + vanilla JS (no framework)
- GSAP + ScrollTrigger via CDN for the 2 cinematic moments
- Canvas-based frame scrubbing for the hero video per `SKILL3D.md` patterns (preload frames, `requestAnimationFrame`, passive scroll listener, devicePixelRatio scaling)
- Google Fonts preconnect + `font-display: swap`
- Mobile-first responsive, breakpoints at 640px and 960px

### Performance targets

- Lighthouse 90+ on all metrics (desktop)
- LCP under 2.5s on mid-tier mobile
- Lazy-load all non-critical imagery (`loading="lazy"`)
- `prefers-reduced-motion` respected on both cinematic moments (fall back to a single still + crossfade)
- `will-change` hints only on elements actively animating

## Open Questions & Risks

1. **Risk — asset generation bottleneck.** User has confirmed sufficient Nano Banana + Seedance credits, so this is unlikely to block, but: Moment 2's brand-story stills are the single largest dependency — if any of the 3 documentary stills comes out off-tone, the cinematic moment falls flat. Mitigation: generate all 3 in one Nano Banana session with a shared palette/mood reference and pick the best.
2. **Risk — hero video first-frame contract.** Per `SKILLassets.md`, first frame must be ≥95% pure white or the hero loader blend breaks. User should verify with the FFmpeg `signalstats` snippet in SKILLassets before handing the video to the build.
3. **Risk — before/after heel imagery.** Easy to end up either too clinical (feels like a medical ad) or too glossy (feels like retouched stock). Generation prompts must emphasise "tasteful editorial beauty shoot, honest, not retouched to perfection."
4. **Decision deferred — typography final pair.** Defaulting to Instrument Serif + Inter + JetBrains Mono. User can swap during build if a stronger pair is identified (Canela, GT Sectra, etc., would also work for the display serif).
5. **Decision deferred — Foot Repair PDP URL slug.** Default: `native-nature/products/miracle-foot-repair-cream.html`. Can change during build if a shorter slug reads better.
